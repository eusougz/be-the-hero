# Be The Hero

### Pacotes Instalados

##### NODEMON

npm install nodemon -D	// -D: especifica no package.json que nodemon é um dependecia apenas para Dev (enquanto está em desenvolvimento)


##### QUERY BUILDER (KNAX.js)

// http://knexjs.org/
// ver 'Installation'

npm install knex
npm install sqlite3

// Depois de instalado, executar

npx knex init

// Lembrando: npx != npm. npx somente EXECUTA o pacote e não instala ele nas dependências do projeto
